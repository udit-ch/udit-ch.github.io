How Networking Makes This Site Work

01

DNS Resolution

DNS resolves udit-ch.github.io/network-project1 to an IPv4 address through the DNS hierarchy.

nslookup · dig · A records

02

IP Addressing

IPv4 addresses provide load balancing across GitHub Pages' edge network. Anycast routing ensures optimal performance.

IPv4 · Load balancing · Anycast

03

HTTP/HTTPS Protocols

All traffic served over HTTPS (HTTP/2 200). HTTP requests return a 308 Permanent Redirect. HSTS header enforced.

HTTP/2 · 200 OK · 308 Redirect

04

Security

HTTPS enforced via 308 redirect. HSTS active with security directives. TLS certificate provisioned by Let's Encrypt.

HSTS · TLS · Let's Encrypt
