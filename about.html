About This Project

CSC 4220 · Computer Networks · Georgia State University · Spring 2026

Project Goal

This website was built as a live demonstration of real-world networking concepts for CSC 4220 at Georgia State University. Every feature — from HTTPS encryption to lazy-loaded images — directly maps to a networking or security concept covered in the course.

The site is publicly deployed at udit-ch.github.io and fully verifiable with browser developer tools and command-line utilities like nslookup, dig, and curl.

How to Verify

You can verify all networking and security features using these commands:

# DNS lookup
nslookup udit-ch.github.io

# IPv4 and IPv6 records
dig udit-ch.github.io A
dig udit-ch.github.io AAAA

# Check HTTPS headers
curl -I https://udit-ch.github.io

# Verify TLS certificate
openssl s_client -connect udit-ch.github.io:443
